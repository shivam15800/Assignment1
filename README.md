# Data Cleaning and Exploration Assignment

This project contains basic data cleaning and exploration performed on the provided Myntra dataset using Python and Pandas.

## Tasks Performed

* Dataset exploration using head(), shape, info(), columns, and dtypes
* Checked missing values
* Filled null values in required columns
* Checked duplicate records
* Applied filtering conditions
* Cleaned `final_price` column by removing currency symbols and converting it to numeric format
* Created derived column `total_amount`
* Saved cleaned dataset as CSV

## Technologies Used

* Python
* Pandas
* Jupyter Notebook

## Files Included

* `demo.ipynb`
* `cleaned_combined_dataset.csv`
* Summary/Report file

This assignment was completed as part of Data Engineering training tasks.
